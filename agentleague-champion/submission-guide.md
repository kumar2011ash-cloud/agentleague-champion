# Submission Guide for AgentLeague Champion

## Core submission requirements

1. Working project hosted in a public GitHub repository
2. Project description explaining features, functionality, and AI value
3. Demo video (2 minutes max) with a public YouTube or Vimeo link
4. Architecture diagram and service mapping
5. Innovation Studios profile activated and team members registered

## Recommended assets for submission

- `README.md` with setup and usage instructions
- `project-description.md` with pitch and value proposition
- `architecture.md` showing how Azure and Foundry are used
- Screenshots of the UI and generated plan
- Demo video script using the generated demo flow

## Demo recording checklist

- Start with the problem statement
- Show the AgentLeague Champion UI
- Generate a project plan from a real idea
- Highlight the architecture and Azure integration
- Close with the submission checklist and why the idea wins

## Strong submission tips

- Keep the demo under two minutes
- Use simple, clear language
- Focus on the AI value and what the agent automates
- Show the actual generated plan in the UI
- Explain how this solution uses Microsoft Foundry, Azure OpenAI, or Microsoft 365

## Deployment ideas

- Host the frontend/backend on Azure App Service or Azure Static Web Apps
- Store prompt history and generated plans in Azure Cosmos DB
- Use Azure Functions for future workflow automation
- Add Microsoft 365 connectors if the idea includes enterprise agent capabilities
